UC7 – Deque-Based Optimized Palindrome Checker

This use case implements a palindrome checker using a Deque (ArrayDeque) in Java.
Characters of the input string are inserted into the deque.
The program removes and compares the first and last characters iteratively.
If all characters match, the string is identified as a palindrome.
This approach avoids string reversal and runs in O(n) time complexity.
