# PalindroneCheckerAPP
Palindrone Checker Management System
📖 Project Objective

The objective of this project is to develop a Java-based Palindrone Checker Management System.

This system is designed to:

Understand Java project structure

Implement a clear application entry point

Display system initialization messages

Build a structured foundation for future palindrome logic

Practice Git version control

The current version focuses on establishing the startup flow of the application.
